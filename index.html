Rails 检查 render 文件路径

发表于 2016-11-24

在代码里使用 render 方法引入 html file json… 如果有路径不存在，就会报错。一般我们引入的文件，路径是静态的，所以不会出现什么问题。
如果我们需要动态引入一个文件，我们可以使用

lookup_context.find_all('file_path')

这个方法，的返回值，如果文件存在，返回文件路径，如果不存在，返回一个空数组。如果在方法后加上 any? 如果不存在 则返回 false.
